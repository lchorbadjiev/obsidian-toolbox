# Quickstart: Book Index Generation Prompt

## Scenario: Generate a book index from saved highlights

**Precondition**: A directory of highlight markdown files saved by `save_highlights`, e.g.:
```
~/notes/invention-of-science/
├── 001 - Human History and the Neolithic Revolution.md
├── 002 - Ancient Admiration for Archimedes.md
└── ...
```

### Via MCP (agent workflow)

1. Agent calls `generate_book_index` prompt with `directory = "~/notes/invention-of-science"`.
2. MCP server returns a `UserMessage` containing all highlight data grouped by chapter,
   plus instructions to generate the index format.
3. Agent generates the index markdown from the prompt.
4. Agent saves the result as `The Invention of Science - Index.md` in the notes folder.

### Via CLI (inspect / pipe)

```bash
otb md index-prompt ~/notes/invention-of-science
# Prints the prompt text to stdout

otb md index-prompt ~/notes/invention-of-science > /tmp/prompt.txt
# Save prompt to file for inspection
```

### Error cases

```bash
otb md index-prompt /nonexistent/path
# Error: Path does not exist (Click validation)

otb md index-prompt /tmp/empty-dir/
# Prints: informative message that no highlights were found
```

## Expected Index Format

```markdown
---
tags: [book, history-of-science]
author: David Wootton
published: 2015
---

# The Invention of Science: A New History of the Scientific Revolution

{prose summary generated by the LLM}

---

## Notes by Chapter

### 1. Modern Minds
- [[notes/001 - Human History and the Neolithic Revolution]]
- [[notes/002 - Ancient Admiration for Archimedes]]

### 2. The Idea of the Scientific Revolution
- ...
```
